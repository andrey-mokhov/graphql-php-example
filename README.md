# Examples for GraphQL library on Spiral Framework

## install

For install example project run:

Cloning repository
```shell
git clone git@github.com:andrey-mokhov/graphql-php-example.git
```

change branch
```shell
git checkout spiral
```

run docker
```shell
make install
```

## other times

start docker
```shell
make up
```

stop docker:
```shell
make down
```

## other commands

restart docker
```shell
make r
```

run shell
```shell
make rr
```

rebuild
```shell
make build
```
