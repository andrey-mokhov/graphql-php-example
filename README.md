# Examples for GraphQL library on Spiral Framework

## install

For install example project run:

```shell
$ make install
```

## other times

start docker
```shell
$ make up
```

stop docker:
```shell
$ make down
```

## other commands

restart dockers
```shell
$ make r
```

run shell
```shell
$ make rr
```

rebuild
```shell
$ make build
```
